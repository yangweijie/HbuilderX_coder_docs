# HbuilderX_coder_docs

程序员手册

## 功能

### 配置当前查询的手册

### 当前手册里关键词搜索后展示手册内容
当前是将html 走接口 转成md 格式 hbuilderx 自带打开md渲染。格式会有丢失。
 
 所有的功能都在 打开文件时 右键出的菜单
